🧩 2. Algoritmo de merge de entidades (el que necesitás)
Este es el error que hoy te rompe el flujo: estás sobrescribiendo entidades en vez de fusionarlas.
Usá este merge:
function mergeEntities(prev, next) {
  const merged = { ...prev };

  for (const key of Object.keys(next)) {
    const value = next[key];

    // Si viene null o undefined, ignorar (Claude a veces manda null)
    if (value === null || value === undefined) continue;

    // Si es una contradicción, reemplazar por la nueva
    merged[key] = value;
  }

  return merged;
}


Y en tu botEngine:
conversation.entities = mergeEntities(conversation.entities, newEntities);


Esto garantiza:
- No se borran entidades previas
- No se pisa con null
- Se actualiza si el usuario corrige algo
- Se mantiene memoria conversacional real

📅 3. Cálculo automático de fechas relativas
Agregá esta función:
function resolveRelativeDates(entities, now = new Date()) {
  if (entities.fechaEntrada === "mañana") {
    const d = new Date(now);
    d.setDate(d.getDate() + 1);
    entities.fechaEntrada = d.toISOString().slice(0, 10);
  }

  if (entities.fechaEntrada && entities.noches && !entities.fechaSalida) {
    const d = new Date(entities.fechaEntrada);
    d.setDate(d.getDate() + entities.noches);
    entities.fechaSalida = d.toISOString().slice(0, 10);
  }

  return entities;
}
