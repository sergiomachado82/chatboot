Luego en botEngine:

const agent = await getAgentConfig()

const prompt = buildPrompt({
   agentName: agent.nombre_agente,
   tone: agent.tono,
   emojis: agent.usar_emojis
})

1. Horarios de atención del bot ⏰

Muy importante.

Permite definir cuándo el bot responde automáticamente.

Ejemplo:

Lunes a Viernes: 08:00 - 22:00
Sábado: 09:00 - 20:00
Domingo: cerrado

Comportamiento:

Dentro de horario

bot responde normalmente

Fuera de horario

mensaje automático:

Hola! Gracias por escribir a Luminar 😊

Nuestro horario de atención es de 8:00 a 22:00.
Igualmente puedo ayudarte con disponibilidad y precios.

Configuración:

horario_inicio
horario_fin
mensaje_fuera_horario
2. Mensaje de bienvenida 👋

Se envía cuando el usuario inicia conversación.

Ejemplo:

Hola! Soy Sofía, asistente de reservas de Luminar.

Puedo ayudarte con:
• disponibilidad
• precios
• fotos de los departamentos

¿Para qué fecha querés alojarte?

Esto mejora mucho la conversión.

Configuración:

mensaje_bienvenida
activar_bienvenida (true/false)
3. Política de reservas automática 🏨

Define qué puede hacer el bot sin humano.

Opciones típicas:

crear_pre_reserva_auto
crear_reserva_auto
solo_consultas

En tu caso ya aplicaste buena práctica:

bot solo crea pre_reserva

Configurable:

auto_pre_reserva = true
4. Reglas de escalamiento a humano 👩‍💻

Evita que el bot maneje situaciones sensibles.

Triggers comunes:

queja
pago
transferencia
problema
cancelación

Ejemplo:

Si detecta "reclamo" → pasar a humano

Configuración:

escalar_si_pago
escalar_si_queja
escalar_si_cancelacion
5. Control de envío de fotos 📷

Muy útil en alojamientos.

Opciones:

enviar fotos automáticamente
solo si usuario pide fotos
no enviar fotos

Configuración:

modo_envio_fotos
auto | on_request | off
6. Configuración del tono del agente 💬

Define personalidad.

Ejemplo:

formal
amigable
informal
premium

Esto modifica el prompt del LLM.

Ejemplo prompt:

Responde de manera amigable, clara y breve.

Configuración:

tone
emoji_level
7. Control de respuestas largas 📏

Evita que el LLM escriba demasiado.

Configuración:

longitud_respuesta

Ejemplo:

corta
media
detallada

Esto reduce tokens y mejora UX.
